Instructions

Cifrar

  • Clique no botão de Cifrar para cifrar sua mensagem
  • Você será direcionado ao campo onde deverá inserir o deslocamento e a mensagem
  • Após inserir os dados corretamente, a mensagem cifrada aparecerá logo abaixo no campo 'Sua mensagem cifrada'

Importante:

  • O DESLOCAMENTO corresponde a uma chave numérica que você escolhe, este número será usado como uma chave de segurança para codificar sua mensagem
  • Guarde o número do DESLOCAMENTO, pois será necessário informá-lo para decodificar sua mensagem

Descifrar

  • Clique no botão de Descifrar para descifrar sua mensagem
  • Você será direcionado ao campo onde deverá inserir o deslocamento e a mensagem
  • Após inserir os dados corretamente, a mensagem descifrada aparecerá logo abaixo no campo 'Sua mensagem cifrada'

Importante:

  • O DESLOCAMENTO corresponde a uma chave numérica que foi escolhida pela pessoa que cifrou a mensagem, e você precisará informá-la corretamente para descifrar a mensagem.
